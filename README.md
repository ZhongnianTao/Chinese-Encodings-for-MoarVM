# Chinese Encodings for MoarVM

This repository stores all the files used in developing GB2312 and GB18030 encoding for MoarVM.

My blog post about GB2312 encoding: [Implementing the GB2312 Encoding](https://zhongniantao.wordpress.com/2019/07/15/implementing-the-gb2312-encoding/).

Here are my PRs for GB2312: [MoarVM (initial commit)](https://github.com/MoarVM/MoarVM/pull/1147), [MoarVM (conversion table compressed)](https://github.com/MoarVM/MoarVM/pull/1150), [Rakudo](https://github.com/rakudo/rakudo/pull/3079), [Roast](https://github.com/perl6/roast/pull/563). 

My PRs for GB18030: [MoarVM](https://github.com/MoarVM/MoarVM/pull/1160), [Rakudo](https://github.com/rakudo/rakudo/pull/3135), [Roast](https://github.com/perl6/roast/pull/569).
